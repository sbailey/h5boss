.. _install:

Installation
=================

Download::

    git clone https://github.com/valiantljk/h5boss.git

Configure::

    cd h5boss/
    export PYTHONPATH=`pwd`/h5boss_py:$PYTHONPATH
    export PATH=`pwd`/h5boss_py/script:$PATH
    module load python/2.7-anaconda (on Edison/Cori, You need astropy, etc)
